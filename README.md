PixelFrame – Image Dimension Inspector

PixelFrame is a browser extension that allows you to instantly view image dimensions with a single click, without being redirected by hyperlinks. Designed for web developers, designers, and professional teams, it provides a clean overlay showing image width and height.

Features:

- Instantly shows image dimensions on click
- Prevents navigation for images wrapped in links
- Supports lazy-loaded images
- Displays dimensions as a clear overlay: width: 120px - height: 100px
- Highlights the image with a neon border for easy identification
- Only one image selected at a time
- Toolbar toggle to activate or deactivate the extension

Installation via Repository:

1. Clone or download the repository to your local machine:
git clone https://github.com/ashkanmostofi/pixelframe-extension.git


or download as ZIP and extract it.

2. Open your browser (Chrome or Edge) and go to the extensions page:
- Chrome: `chrome://extensions`
- Edge: `edge://extensions`

3. Enable **Developer Mode** in the top-right corner.

4. Click **Load Unpacked** and select the folder containing the extension.

5. The PixelFrame icon will appear in the toolbar. Use it to toggle the extension on or off.

6. Click on any image in the page to see its dimensions displayed.

Note: For public distribution without Developer Mode, you need to publish the extension on the Chrome Web Store.
